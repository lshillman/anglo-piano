Keyboard shortcuts

[left] / [right] : select the previous / next piano key

[1] - [5] : select chords

[shift] + click : select multiple notes on either keyboard

[esc] : close dialog boxes like this one

Editor:

[up] / [down] : transpose the current note up / down

[shift] + [left] / [right] : move the current button left or right

[shift] + [backspace] : delete the current button
